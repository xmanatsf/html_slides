  • GPUs = "long-haul trucks"; LPUs = "delivery vans" — single architecture is suboptimal

  • Connected via NVLink (intra-rack) and Spectrum-X (inter-rack)
  • Targets 25% of AI use cases (ultra-fast inference); Vera Rubin addresses 75%

First fully NVDA-designed CPU optimized for AI/agentic workloads. ~2x performance vs Grace; ~50% higher per-core performance and ~3x more memory per core than x86. CPUs re-emerging as gating factor for inference: handle tool execution, code compilation, validation, storage orchestration, workflow control. NVDA does not view LPX or standalone CPUs as margin-dilutive; GM not expected above 75%.
